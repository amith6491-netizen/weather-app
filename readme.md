🌦️ Weather App

A simple and responsive Weather Web Application built using HTML, CSS, and JavaScript.
This project fetches live weather data from a public API and displays the current weather and forecast for any city.

📍 A beginner-friendly project ideal for learning API integration and DOM manipulation.

🚀 Features:

✔️ Search weather by city name
✔️ Displays city, country, and local weather
✔️ Shows temperature, humidity, and weather description
✔️ Responsive UI
✔️ Uses live data from a weather API
✔️ Minimal, clean design

⚙️ How It Works:

User enters a city name in the input.
JavaScript fetches data from the Weather API (e.g., OpenWeather).
The weather data is parsed and displayed on the UI:
Temperature
Weather description
Humidity
City and country
